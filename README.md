# antiGGGravity MEPF — Revit Add-in Installer

**antiGGGravity MEPF** is a professional Revit add-in toolkit for MEPF engineers, supporting Revit 2022 through 2027.

## Download

Download the latest installer ZIP from the [Releases](../../releases) page or directly from this repository.

**Current Version:** 1.0.0

## Installation

1. Close Revit.
2. Download `antiGGGravity_MEPF_Installer_AllVersions.zip` and extract it.
3. Double-click `install.bat`.
4. The installer automatically detects all compatible Revit versions (2022–2027) installed on your system.
5. Launch Revit — you will see the **antiGGGravity MEPF** tab in the ribbon.

## Uninstallation

Run `uninstall.bat` to remove the add-in from all detected Revit versions.

## Supported Revit Versions

| Revit Version | Framework       |
|---------------|-----------------|
| 2022          | .NET 4.8        |
| 2023          | .NET 4.8        |
| 2024          | .NET 4.8        |
| 2025          | .NET 8 (Windows)|
| 2026          | .NET 8 (Windows)|
| 2027          | .NET 10 (Windows)|

## License Activation

A license key is required to use antiGGGravity MEPF tools.

1. Open the **Hardware ID** tool in the **antiGGGravity MEPF** License panel.
2. Copy your unique Hardware ID.
3. Email it to [antiGGGravity.info@gmail.com](mailto:antiGGGravity.info@gmail.com) to request a trial or purchase a license.
